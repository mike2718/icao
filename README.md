# 国际民航组织字母解释法翻译器C程序

将一串呼号转换成国际民航组织字母解释法的一系列单词组合。

## 例子

```
mike@hp2540p:~/dev2/icao/ > clang -o icao icao.c && ./icao
请输入一个呼号：bg7xtq
呼号bg7xtq的字母解释法是：
 Bravo   Golf    Seven   X-ray   Tango   Quebec  Echo    Tango   Uniform         Tango        Uniform         Papa    Tango   Uniform         Nine    Five

```
